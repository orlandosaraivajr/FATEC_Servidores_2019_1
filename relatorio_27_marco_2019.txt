Alias ( Apelidos )

Criar apelidos para os comandos
como ?

alias 'listar'='ls -la'

'listar' => chave
'ls -la' => valor
==========================================
Apelidos
========
Formas de inserir apelidos:
Sessão corrente -> 
Funciona apenas na sessão corrente

Apenas para o usuário -> 
Editar o arquivo oculto .bashrc ( localizado no diretório do usuário )

Para todos os usuários do sistema -> 
Editar o arquivo /etc/bash.bashrc ( SOMENTE O ROOT PODE )

==========================================
